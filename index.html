About

Built by a developer
who needed these tools

I'm Drew, a developer and IT team lead. I kept needing Claude to see things happening on my screen and felt like there had to be a better way than tediously taking screenshots of each step. So I built a tool where I could record processes, narrate what's happening, and hand it off to Claude directly. It worked so well that I realized this was something other AI power users and developers would want too.

KBGen comes from years of managing an IT support team where good documentation is a must. But the reality is that the IT and tech space is only getting busier. KBGen is built to simplify your workflows, create better documentation faster, and stay compliant with standards like HIPAA and SOC2. Less time writing docs, more time solving problems.
